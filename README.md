# Master's Thesis
The `source.sh ` file contains script for using Pandoc to convert content into HTML or PDF.

- `pandoc`
- `texlive`
- `texlive-xetex`
- `texlive-lang-european`
- `inotify-tools`

We can use the script by sourcing it.

```bash
. source.sh
```

The script contains the following functions.

- `html`
- `pdf`
- `preview`
